# **App Name**: Friendship Chronicle

## Core Features:

- Member Profile Management: Store and manage member information including name, phone number, address, birthday (with lunar/solar option), anniversary, food preferences, and hobbies.
- Anniversary Calculation: Automatically calculate major anniversary dates (61st, 70th, 80th birthdays) based on stored birth years.
- Multimedia Gallery: Upload and share photos from past meetups or travel recommendations. Utilize Firebase Storage for high-quality image storage.
- Inspirational Messages: Create a space for sharing inspirational quotes or messages, potentially compiled into a 'group anthology' later.
- Digital Rulebook: Easily accessible digital version of the group's rulebook, eliminating the need for paper copies.
- Smart Digital Directory: Searchable directory with contact information, address, and direct call functionality.
- AI Content Summarization Tool: Summarize a set of inspirational messages to find key insights, or topics, as the first step towards generating an anthology.

## Style Guidelines:

- Primary color: Deep violet (#9400D3) to represent wisdom and respect among members.
- Background color: Light purple (#E6E0EB), a desaturated version of the primary color for a calm backdrop.
- Accent color: Blue violet (#8A2BE2) used for interactive elements.
- Body and headline font: 'Alegreya', a serif font that will impart a literary, sophisticated touch to all the content.
- Clean, modern icons for navigation and features.
- Simple, card-based layout for easy readability and navigation.
- Subtle transitions and animations to enhance user experience.
